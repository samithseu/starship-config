## Requirement

- Nerd Font: <https://www.nerdfonts.com/font-downloads>

## Installation

### Windows

```powershell
iwr "https://github.com/samithseu/starship-config/raw/main/starship.toml" -OutFile $HOME/.config/starship.toml
```

### Linux & MacOS

```bash
curl -L -o "$HOME/.config/starship.toml" "https://github.com/samithseu/starship-config/raw/main/starship.toml"
```
